## About  
A React-based cryptocurrency tracker that displays real-time data like price, volume, market cap, and 24-hour trends. It includes search and currency conversion features.

## Tech Stack  
- React  
- JavaScript  
- CSS  
- Axios  
- Chart.js  
- React Router v6
